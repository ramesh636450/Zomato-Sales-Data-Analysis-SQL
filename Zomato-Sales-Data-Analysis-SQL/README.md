# Zomato Sales Data Analysis – SQL

## Project Overview
Performed SQL-based analysis on food delivery sales data to identify customer trends, restaurant performance, and revenue insights.

## SQL Concepts Used
- Joins
- CTEs
- Window Functions
- Aggregations
- GROUP BY
- Subqueries

## Key Analysis
- Top-performing restaurants
- Revenue trends
- Customer spending behavior
- Order analysis

## Business Insights
Generated insights to improve customer targeting and restaurant performance analysis.
